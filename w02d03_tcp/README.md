### OSI (Open Systems Interconnection) Layers in Networking

#### Interesting to know
- Physical layer (ethernet, usb, coax)
- Datalink layer (wifi)
- Network layer (IP)

IP address: 127.0.0.1 (localhost)
ipv6 ::1

#### Be aware of
- Transport layer (TCP, UDP)
- Session layer (*TCP)
- Presentation layer (SSL, TLS)
- Application layer (HTTP, SMTP, SSH)
